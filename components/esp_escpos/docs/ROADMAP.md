# Roadmap

This project is currently focused on the core receipt-printing path:

- USB ESC/POS printing
- text formatting
- receipt columns
- local EAN13 rendering
- local QR rendering
- BMP image printing
- raw ESC/POS commands
- feed, cut, and beep helpers

Likely future improvements:

- more QR versions and error-correction modes
- more locally rendered barcode formats
- PNG/JPG decoding through optional image decoder integrations
- optional UART or network transports
- more printer profile examples

For current usage, start with `README.md` and `API.md` in this folder.
